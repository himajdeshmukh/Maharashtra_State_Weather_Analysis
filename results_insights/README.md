# Analysis Results and Insights

This directory contains the final non-graphical outputs generated from
the Maharashtra climate data analysis pipelines.

All files in this folder are **automatically generated** when analysis
scripts are executed and are not manually edited.

## Contents

### Excel Reports (`.xlsx`)
These files contain structured summary tables and comparison results,
intended for further analysis, reporting, and review.

Typical contents include:
- Decade-wise climate summaries
- Region-wise recent climate statistics
- Climate change impact assessments
- Significance summary tables
- Monsoon analysis tables (where applicable)

### Insights Reports (`.txt`)
These text files translate numerical results into interpretable findings,
highlighting long-term climate trends, regional contrasts, and
agricultural or environmental implications.

Typical contents include:
- Executive summaries
- Key climate findings
- Region-wise interpretations
- Monsoon behavior insights
- Actionable recommendations

## Usage Notes
- Files are timestamped to avoid overwriting previous results
- This folder is typically excluded from version control using `.gitignore`
- Visualizations are stored separately in the `visualisation/` directory
- Contents vary depending on the analysis scenario:
  - All regions included
  - Konkan region excluded
